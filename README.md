msu3-filters
============

First task for the CG course. Implements a set of image filters.<br>
Win32 only, but running under WINE is also fully supported.

(original description in Russian:)

Задание:               1 / Редактор изображений

Базовая часть:<br>
Автоуровни (-l)       [+]<br>
Автоконтраст (-c)     [+]<br>
"Серый мир" (-a)      [+]<br>
Оттенки серого (-x)   [+]<br>
Медиана (-m)          [+]<br>
Гаусс (-g)            [+]<br>
Собель по Y (-s)      [+]<br>
Линейное преобр. (-k) [+]<br>
Поворот (-r)          [+]<br>
Масштабирование (-z)  [+]<br>

Дополнительная часть:<br>
Собственный GUI       [+]<br>
Выделение региона     [+]<br>
Сепарабельный Гаусс   [+]<br>
Распараллеливание     [+]<br>
Эффект волн (-w, -h)  [+]<br>
Эффект стекла (-f)    [+]<br>
Фильтр "Плитка" (-t)  [+]<br>

Система:               MinGW / Code::Blocks 10.05<br>
ОС:                    Windows XP SP2, Windows 7 Home Extended, WINE 1.5.12-1<br>
Аппаратура:            Core2Duo T5500 1.66 ГГц, 2048 МБ, Radeon X1700 Mobility<br>

Комментарии:<br>
Вместо Qt использовались библиотеки Win32 API.<br>
Поддерживаемые входные форматы  - BMP, JPG, GIF (а из-под WINE - ещё и PNG).<br>
Поддерживаемые выходные форматы - BMP.<br>
Фильтры Собеля и линейной трансформации считают, что за краем картинки - нули.<br>
Остальные фильтры дополняют картинку значениями крайних пикселей.<br>
Дополнительный декоративный фильтр "Плитка" реализует эффект стеклянной плитки.<br>
При запуске без параметров, выполняется GUI-версия программы, иначе - CLI.<br>

(Про GUI)<br>
1. Можно одновременно настраивать несколько фильтров.<br>
2. Присутствует возможность отмены и повтора действий над изображением.<br>
3. Ухватившись за картинку указателем мыши, её можно таскать с места на место.<br>
4. Во время выделения изображения рамкой, концевые маркеры могут "моргать".<br>
5. Выделять прямоугольник после нажатия кнопки "Выделить область" можно 1 раз.<br>
6. Изменять регион возможно, только если нажать кнопку ещё раз и перевыделить.<br>
7. Фильтры считают, что регион = отдельная картинка, и за его границами пусто.<br>
8. Цветная полоса сбоку от настроек фильтра содержит цвет его рамки выделения.<br>

(Про CLI)<br>
1. Формат: filters.exe -@ аргумент вход выход (где вместо @ - буква ключа).<br>
2. Принимаются только односимвольные ключи, имена идентичны именам в задании.<br>
3. Разделителями параметров считаются пробелы (\x20).<br>
4. Пробел между ключом и его аргументом не обязателен.<br>
5. Количество пробелов подряд не критично, везде, кроме как внутри параметров.<br>
6. Чтение параметров, заключённых в кавычки, не поддерживается.<br>
7. Исключение из [5] и [6] - строка с матрицей для линейного фильтра.<br>
8. Коэффициент в фильтре масштабирования задаётся не в %, а в разах.<br>
9. Параметр эффекта "Волны" - длина волн, параметр "Стекла" - радиус.<br>
